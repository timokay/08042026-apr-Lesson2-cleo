# Jobs-to-be-Done Canvas

Шаблон для Module 2. Для каждого сегмента заполни все 3 уровня задач.

## Framework

```
┌──────────────────────────────────────────────────────────┐
│  CUSTOMER SEGMENT: [Название]                             │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  FUNCTIONAL JOB (что хочу СДЕЛАТЬ):                      │
│  "Когда я [ситуация], я хочу [действие],                │
│   чтобы [результат]"                                     │
│                                                          │
│  EMOTIONAL JOB (что хочу ЧУВСТВОВАТЬ):                   │
│  "Я хочу чувствовать [эмоция]"                           │
│  "Я хочу избежать чувства [эмоция]"                      │
│                                                          │
│  SOCIAL JOB (как хочу ВЫГЛЯДЕТЬ):                        │
│  "Я хочу, чтобы другие видели меня как [образ]"         │
│  "Я не хочу выглядеть как [негативный образ]"           │
│                                                          │
├──────────────────────────────────────────────────────────┤
│  CURRENT SOLUTION:                                       │
│  [Что используют сейчас] → Тратят: $X/мес, X часов/нед  │
│                                                          │
│  SWITCHING BARRIERS:                                     │
│  [Что мешает переключиться: привычка, данные, контракт]  │
│                                                          │
│  TRIGGER EVENT:                                          │
│  [Что заставляет начать искать новое решение]            │
└──────────────────────────────────────────────────────────┘
```

## Примеры для Noom

**Сегмент 1: "Хронический диетчик"**
- Functional: "Когда я в очередной раз набираю вес после диеты, я хочу понять ПОЧЕМУ я переедаю, чтобы сломать цикл йо-йо"
- Emotional: "Хочу чувствовать контроль, а не вину"
- Social: "Хочу выглядеть как человек, который заботится о себе, а не как вечно сидящий на диете"
- Trigger: Фото на событии, визит к врачу, новый год

**Сегмент 2: "Осознанный оптимизатор"**
- Functional: "Когда я хочу улучшить здоровье системно, я хочу data-driven подход, чтобы видеть прогресс по метрикам"
- Emotional: "Хочу чувствовать себя экспертом своего тела"
- Social: "Хочу выглядеть как biohacker, а не как больной"
- Trigger: Wearable data показал плохие метрики
