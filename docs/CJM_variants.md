# 🗺️ Customer Journey Map: Клёво (Cleo RU Clone)
**Дата:** 2026-04-08 | **Целевой рынок:** Россия, Gen Z 18-30 лет
**Продукт:** AI-финансовый ассистент с характером для российского рынка

---

## Контекст: Извлечённые CJM Building Blocks

```yaml
company: Клёво (Klyovo — "круто/классно" на молодёжном сленге, звучит как Cleo)
one_liner: "Дзен-мани + саркастичный характер на AI" — финансовый советник, который не скучный
segments:
  - name: "Транжира-Осознанный"
    functional_job: "Помоги мне понять, куда уходят деньги"
    emotional_job: "Хочу чувствовать контроль без стресса"
    social_job: "Хочу выглядеть как человек, который управляет финансами"
    trigger: "Конец месяца — деньги кончились, а почему — непонятно"
  - name: "Подписки-Паразиты"
    functional_job: "Найди мне лишние траты, которые я не замечаю"
    emotional_job: "Хочу почувствовать, что не теряю деньги зря"
    social_job: "Хочу похвастаться, что оптимизировал расходы"
    trigger: "Пришла выписка, увидел 5 подписок, о которых забыл"
  - name: "Зарплата-до-Зарплаты"
    functional_job: "Помоги дожить до зарплаты"
    emotional_job: "Хочу избавиться от финансовой тревоги"
    social_job: "Не хочу просить в долг у родителей"
    trigger: "Осталось 5 дней до зарплаты, денег нет"
aha_moment: "AI называет тебя по имени и показывает, что ты потратил на кофе больше, чем на аренду"
complaints_from_existing_tools:
  - "Дзен-мани скучный — это просто таблица Excel в красивой упаковке"
  - "CoinKeeper требует вводить каждую трату вручную — надоедает за неделю"
  - "Банковские AI-боты отвечают шаблонами"
positive_patterns:
  - Юмор и самоирония про расходы вирален в ВКонтакте
  - Люди охотно делятся "стыдными" тратами в мемах
  - Практические финансовые советы в Telegram наборирают тысячи просмотров
pricing_tiers:
  - free: CSV upload + анализ трат + ростер + топ-5 категорий
  - plus: 299 руб/мес — автосбережения + подключение банков + история
why_now:
  - Западный финтех ушёл из РФ (Cleo недоступна)
  - Gen AI рынок РФ вырос до 58 млрд руб в 2025
  - Молодёжь финансово уязвима (импульсивные покупки + низкая грамотность)
  - Т-Банк/Сбер не заполнили нишу "personality-driven finance"
```

## 📊 Таблица сравнения 3 вариантов CJM

| | **A: "Честный Ростер"** | **B: "Финансовый Друг"** | **C: "Паразит-Детектор"** |
|---|---|---|---|
| **Aha Moment** | AI ростит тебя с юмором за конкретные траты ("ты потратил 4200₽ на доставку суши за неделю — это как ужин в ресторане каждый день, только хуже") | AI помнит твои цели и радуется прогрессу как друг ("Ты на 15% ближе к поездке на Алтай!") | AI находит скрытые расходы ("Я нашёл 3 подписки, о которых ты забыл — ₽1,840/мес утекает") |
| **Entry Hook** | "Дай AI поджарить твои расходы" (Roast Mode) | "Расскажи AI о своей мечте — он поможет дойти до неё" | "За 2 минуты найдём деньги, которые ты уже теряешь" |
| **Onboarding** | Загрузка CSV или ввод банка → мгновенный ростер в чате | Вопросы о целях → персонализация под мечты | Сканирование трат → поиск паразитов без настройки |
| **Core Loop** | Ростер → смех → делишься → возвращаешься за советом | Дашборд целей → прогресс → AI хвалит → делаешь больше | Экономия найдена → отписка → AI показывает, сколько сэкономил |
| **Paywall Timing** | Сразу после первого ростера (эмоциональный пик) | После 7 дней habit formation | После демонстрации конкретной экономии |
| **Paywall Framing** | "Получи персональный план от AI" | "Открой автопилот для своих целей" | "Подключи банк и экономь автоматически" |
| **Invite Mechanism** | "Поделись своим ростером" (вирусный шеринг скриншотов) | "Пригласи друга — оба получат месяц Plus" | "Покажи другу, сколько он теряет каждый месяц" |
| **Best for Segment** | Транжира-Осознанный (хочет развлечение + осознанность) | Зарплата-до-Зарплаты (хочет систему + мотивацию) | Подписки-Паразиты (хочет немедленную ценность) |
| **Hypothesis** | Вирусность ростера = органический рост | Долгосрочные цели = высокий LTV | Моментальная ценность = высокая активация |
| **Risk** | Ростер может казаться обидным, а не смешным (культурный риск) | Длинный онбординг = высокий отток до aha | Паразиты закончатся — а дальше что? |
| **Estimated Activation Rate** | 45-55% | 30-40% | 60-70% |
| **Estimated Conversion (free→paid)** | 12-18% | 15-22% | 8-12% |
| **Viral Coefficient** | 1.4-1.8 | 0.6-0.9 | 0.8-1.2 |
| **Recommended для российского рынка** | ✅ ОПТИМАЛЬНЫЙ | ⚠️ Вторичный | ✅ Хороший стартер |

## Вариант A: "Честный Ростер" — детальный CJM

> **Рекомендован как основной** — см. ADR-002

### AARRR Framework

```
ACQUISITION → ACTIVATION → RETENTION → REVENUE → REFERRAL
```

### Этап 1: AWARENESS (Осведомлённость)

| Параметр | Детали |
|----------|--------|
| **Точки контакта** | VK-мем "AI поджарил мои траты", Telegram-бот, TikTok-клип с ростером |
| **Эмоция пользователя** | 😂 Смех → узнавание себя |
| **Сообщение** | "ИИ без фильтров скажет тебе правду о твоих деньгах" |
| **Micro-trend** | Finance-as-Culture: [76% Gen Z — соцсети вместо банков](https://www.fintechweekly.com/magazine/articles/how-gen-z-is-changing-personal-finance-habits-in-2025) |
| **Call to Action** | "Узнай, что думает AI о твоих тратах" |
| **KPI** | CTR > 3%, CPL < 80₽ |

### Этап 2: ACQUISITION (Привлечение)

| Параметр | Детали |
|----------|--------|
| **Точка входа** | Лендинг с анимированным ростером (15 сек. демо) |
| **Трение** | Минимальное: "Загрузи CSV или введи 5 трат вручную" |
| **Эмоция** | 🤔 Любопытство — "что AI скажет про меня?" |
| **Micro-trend** | Proactive Finance: [AI знает о тебе больше тебя](https://m2pfintech.com/blog/10-banking-and-fintech-trends-that-will-redefine-2026-and-beyond/) |
| **KPI** | Sign-up rate > 25% от посетителей |

### Этап 3: ACTIVATION — AHA MOMENT ⚡

| Параметр | Детали |
|----------|--------|
| **Триггер** | Первый ростер в чате (генерируется за 10 сек. после загрузки данных) |
| **AHA формулировка** | "Клёво: Слушай, ты потратил ₽12,400 на доставку еды — это больше, чем на коммуналку. Ты кормишь курьеров лучше, чем платишь за жильё 🫠" |
| **Эмоция** | 😱😂 Шок + смех = максимальное вовлечение |
| **Micro-trend** | AI Micro-behavior Detection: [анализ паттернов](https://appinventiv.com/blog/fintech-trends/) |
| **Действие** | Скриншот → шеринг в VK/Telegram → вирусность |
| **Время до AHA** | < 60 секунд от регистрации |
| **KPI** | 55%+ пользователей завершают первый ростер |

### Этап 4: ENGAGEMENT (Вовлечение)

| Параметр | Детали |
|----------|--------|
| **Core Loop** | Ежемесячный ростер → инсайты → советы → прогресс |
| **Фичи Free** | Топ-5 категорий трат, поиск подписок, один ростер/месяц |
| **Engagement механики** | Streak (дни подряд без лишних трат), челленджи, финансовые мемы |
| **Micro-trend** | FHaaS gamification: [геймификация финансового здоровья](https://innowise.com/blog/fintech-trends/) |
| **KPI** | D7 retention > 35%, D30 > 20% |

### Этап 5: REVENUE (Монетизация)

| Параметр | Детали |
|----------|--------|
| **Paywall Trigger** | После первого ростера — "Хочешь план от AI, как реально сократить это?" |
| **Фрейминг** | "Клёво Plus: AI, который не просто ростит, но и чинит" |
| **Цена** | 299₽/мес (эквивалент $3.3 — ниже Cleo за счёт курса) |
| **Что даёт Plus** | Автосбережения, подключение банков (ОФД/СМС), история, неограниченный ростер |
| **Micro-trend** | FHaaS bundled services: [экосистема финансового здоровья](https://innowise.com/blog/fintech-trends/) |
| **KPI** | Free→Plus conversion 15%+ в первый месяц |

### Этап 6: REFERRAL (Реферал)

| Параметр | Детали |
|----------|--------|
| **Механика** | "Поделись своим ростером" — генерация персонального скриншота для шеринга |
| **Инцентив** | Пригласи друга → оба получают месяц Plus бесплатно |
| **Вирусность** | Ростер-скриншот = готовый мем-контент для VK/Telegram |
| **Micro-trend** | Finance-as-Culture viral loops: [Gen Z Women доминируют в Finance TikTok](https://www.newsanyway.com/2026/04/03/gen-z-women-are-quietly-dominating-finance-tiktok-and-changing-the-industry) |
| **KPI** | Viral coefficient K > 1.2 |

## Вариант B: "Финансовый Друг" — детальный CJM

### Этапы

| Этап | Детали |
|------|--------|
| **Awareness** | "AI, который помнит твои цели и болеет за тебя" — эмоциональный контент |
| **Acquisition** | Лендинг с вопросом "Какая у тебя финансовая мечта?" |
| **AHA** | "Ты на 15% ближе к поездке в Грузию! Ещё ₽2,400 — и билеты твои" |
| **Core Loop** | Цель → прогресс → AI хвалит → ставишь новую цель |
| **Paywall** | После 7 дней: "Открой автопилот для своих целей" |
| **Strength** | Высокий LTV (привязка к долгосрочным целям), низкий churn |
| **Weakness** | Долгий онбординг, сложно объяснить value prop быстро |
| **Verdict** | Хорош как второй продукт или upgrade из Варианта A |

## Вариант C: "Паразит-Детектор" — детальный CJM

### Этапы

| Этап | Детали |
|------|--------|
| **Awareness** | "Узнай, сколько денег уходит незаметно" — рациональный крюк |
| **Acquisition** | Лендинг: "2 минуты → найдём скрытые расходы" |
| **AHA** | "Нашли ₽1,840/мес: Netflix (кинул друг, забыл), Canva (автопролонг), Headspace (2021 год)" |
| **Core Loop** | Сканирование → находки → отписка → новые сканирования |
| **Paywall** | После показа конкретной экономии: "Подключи банк, чтобы отслеживать автоматически" |
| **Strength** | Высокая активация (60-70%), немедленная ценность, низкое трение |
| **Weakness** | Одноразовая ценность (паразиты кончаются), слабый долгосрочный retention |
| **Verdict** | Лучший onboarding hook, но нужна Variant A как основной продукт |

## ✅ Рекомендация: Гибридный подход A+C

**Оптимальный вариант для запуска:**

```
Онбординг: C (Паразит-Детектор) → быстрая активация, немедленная ценность
           ↓
Core Product: A (Честный Ростер) → вирусность + регулярное вовлечение
           ↓  
Upgrade Path: B элементы (цели) → как часть Plus плана
```

**Обоснование:**
1. **Паразит-Детектор** как первый экран снижает барьер входа (60-70% активация)
2. **Честный Ростер** как ядро создаёт вирусность и retention
3. **Цели-функции** (В) добавляются в Cleo Plus для удержания

Это решение зафиксировано в [ADR-002](./ADR-002-cjm-variant-selection.md).
