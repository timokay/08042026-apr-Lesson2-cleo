# ADR-004: Монетизационная модель для Клёво

**Статус:** Принято  
**Дата:** 2026-04-08  
**Автор:** Replicate Pipeline (Claude)  
**Контекст:** Phase 1 — бизнес-модель

---

## Контекст

Оригинальная Cleo монетизирует через Freemium ($5.99/мес) + Cash Advance. Для российского рынка нужно адаптировать с учётом:
- Покупательной способности РФ (курс ~90 руб/$)
- Регуляторного контекста (ЦБ РФ)
- Конкурентной среды (бесплатные функции у банков)

## Анализ вариантов монетизации

### Вариант 1: Freemium (как Cleo)

| Параметр | MVP | v1 | v2 |
|----------|-----|----|----|
| Бесплатно | CSV upload + анализ + 1 ростер/мес | + 3 ростера/мес | + история 3 мес |
| Plus (299₽/мес) | Неограниченный ростер | + автосбережения | + cash flow прогноз |
| Pro (599₽/мес) | — | — | + инвестиционные советы |

**Плюсы:** Проверенная модель, низкий барьер входа  
**Минусы:** Нужна масса пользователей для unit economics

### Вариант 2: Подписка без бесплатного уровня (149₽/мес)

**Плюсы:** Простая монетизация  
**Минусы:** Высокий барьер входа для вирусного продукта

### Вариант 3: Pay-per-use (5₽ за ростер, 10₽ за отчёт)

**Плюсы:** Нет commitment, низкий порог  
**Минусы:** Сложно предсказать LTV, мало retention-инцентивов

### Вариант 4: B2B2C (партнёрство с банками)

**Плюсы:** Высокий средний чек, enterprise стабильность  
**Минусы:** Долгий sales цикл, не подходит для MVP

## Решение: Freemium (Вариант 1) с акцентом на early conversion

**Принято:**

### Тарифные планы

**Клёво Бесплатно:**
- Загрузка CSV транзакций (до 100 операций)
- Анализ трат — топ-5 категорий
- Паразит-сканер (поиск лишних подписок) — 1 раз
- Один AI-ростер в месяц
- Шеринг ростера (для вирусности)

**Клёво Plus — 299₽/мес:**
- Неограниченные ростеры и советы в чате
- Подключение банков (API или SMS) — история автоматически
- Автосбережения по правилу (процент от трат)
- Финансовые цели с прогресс-баром
- AI-советы по оптимизации (персонализированные)
- История за 12 месяцев

**Клёво Pro — 599₽/мес (v2):**
- Всё из Plus
- Сравнение с "нормой" (аналогичный доход/регион)
- Прогноз cash flow на 3 месяца
- Базовые советы по инвестициям (ОФЗ, депозиты)

### Ценообразование

| Тариф | Цена РФ | Эквивалент USD | Vs Cleo |
|-------|---------|----------------|---------|
| Бесплатно | 0₽ | $0 | Как Cleo Free |
| Plus | 299₽/мес | ~$3.3 | Cleo Plus $5.99 (в 1.8x дешевле) |
| Pro | 599₽/мес | ~$6.7 | Новый уровень |

**Обоснование цены 299₽:** 
- Это меньше 1 чашки кофе в Старбакс (350₽)
- Среднее ARPU мобильных подписок в РФ: 200-400₽
- [Дзен-мани](https://zenmoney.app/): 99₽/старт; [CoinKeeper](https://a2is.ru/catalog/uchyot-lichnykh-finansov/compare/coinkeeper/zen-money): 149₽/старт

### Cash Advance (v2 — с лицензией ЦБ)

Оригинальная Cleo зарабатывает на Cash Advance (до $250 без процентов, комиссия $3.99).  
Для РФ аналог: **микрокредитная линия** (требует лицензию МФО или партнёрство с лицензированным МФО).

**Решение MVP:** Cash Advance отложить до v2, сосредоточиться на подписочной модели.

## Юнит-экономика (прогноз)

| Метрика | Целевые значения |
|---------|-----------------|
| CAC (стоимость привлечения) | < 150₽ |
| Free → Plus conversion | 12-15% (в течение 30 дней) |
| Monthly Churn (Plus) | < 8% |
| ARPU (Avg Revenue Per User) | 45₽/мес (учёт free-пользователей) |
| LTV (Plus user) | 299₽ / 0.08 = ~3,737₽ |
| LTV/CAC ratio | > 25x |
| Breakeven | 10,000 Plus-подписчиков (~3M₽/мес ARR) |
