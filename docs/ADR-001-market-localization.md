# ADR-001: Локализация продукта для российского рынка

**Статус:** Принято  
**Дата:** 2026-04-08  
**Автор:** Replicate Pipeline (Claude)  
**Контекст:** Клонирование Cleo AI на российский рынок

---

## Контекст

Оригинальный продукт Cleo AI (https://meetcleo.com) работает преимущественно на рынке США. Западные финтех-сервисы (Revolut, Chime, Cleo, Cash App) недоступны в России после 2022 года. Необходимо принять решение о стратегии локализации.

## Анализируемые варианты

### Вариант 1: Минимальная локализация (UI на русском, логика без изменений)
- **Плюсы:** Быстрый запуск, меньше разработки
- **Минусы:** Культурные несоответствия, юмор не работает, интеграция с RU банками сложна

### Вариант 2: Глубокая локализация (продукт пересобран под РФ)
- **Плюсы:** Культурно релевантно, работает с российскими банками
- **Минусы:** Больший объём разработки, нужна локальная команда

### Вариант 3: Частичная локализация (базовая технология + глубокая культурная адаптация)
- **Плюсы:** Баланс скорости и релевантности
- **Минусы:** Требует чёткого определения границ

## Решение: Вариант 3 — Частичная локализация

**Принято на основании:**
1. **Импортозамещение:** Западный финтех ушёл → [Statista: Russia FinTech](https://www.statista.com/topics/6609/fintech-in-russia/) — окно возможностей открыто
2. **Языковой барьер:** Российская молодёжь потребляет финансовый контент на русском (ТЖ, vc.ru) — [Тинькофф Журнал](https://journal.tinkoff.ru/)
3. **Банковская интеграция:** В РФ нет аналога Plaid → нужно SMS-парсинг или Open API банков
4. **Юмор:** "Roast" культура есть в РФ (самоирония в ВКонтакте), но тональность иная — менее агрессивная

## Детали решения

| Аспект | Cleo (оригинал) | Клёво (клон РФ) |
|--------|----------------|-----------------|
| Язык | Английский | Русский |
| Банковская интеграция | Plaid (США) | СМС-парсинг + Открытые API (Т-Банк, Сбер) |
| Валюта | USD | RUB |
| Культурный тон | "Sassy bestie" (американский) | "Честный друг с характером" (российский) |
| Контент | US pop culture references | RU молодёжная культура (мемы, сленг) |
| Юмор | Агрессивный roast | Дружелюбная самоирония |
| Цена | $5.99/мес | 299₽/мес (~$3.3) |
| Название | Cleo | Клёво (klyovo = "классно/круто") |
| Каналы роста | TikTok, Reddit | VKонтакте, Telegram, TikTok |

## Последствия

- **Технические:** Нужен модуль парсинга SMS от российских банков (MVP) + API-интеграции (v1)
- **Продуктовые:** System prompt для Claude/GPT-4 адаптирован под российскую культуру
- **Юридические:** Нужна лицензия на обработку финансовых данных (ФЗ-152, ФЗ-161)
- **Маркетинговые:** Контент-стратегия через ВКонтакте + Telegram + TikTok (не заблокирован в РФ)
