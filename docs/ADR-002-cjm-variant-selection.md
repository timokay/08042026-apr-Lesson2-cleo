# ADR-002: Выбор варианта CJM для продукта Клёво

**Статус:** Принято  
**Дата:** 2026-04-08  
**Автор:** Replicate Pipeline (Claude)  
**Контекст:** Phase 0, Module 2.5 — выбор основного CJM среди 3 вариантов

---

## Контекст

По итогам Module 2 (Product & Customers) были сформированы 3 варианта Customer Journey Map:
- **A: "Честный Ростер"** — вирусная механика, юмор, немедленный AHA
- **B: "Финансовый Друг"** — долгосрочные цели, поддержка, высокий LTV  
- **C: "Паразит-Детектор"** — быстрая активация, немедленная ценность

Нужно выбрать оптимальный вариант для российского рынка Gen Z 18-30.

## Анализ вариантов

### Вариант A: "Честный Ростер"
**Score: 8.2/10**

| Критерий | Оценка | Обоснование |
|----------|:------:|-------------|
| Вирусный потенциал | 9/10 | Self-roast контент вирален: [Gen Z Finance TikTok 2026](https://www.newsanyway.com/2026/04/03/gen-z-women-are-quietly-dominating-finance-tiktok-and-changing-the-industry) |
| Скорость активации | 8/10 | Первый ростер за 60 сек = немедленный AHA |
| Долгосрочный retention | 7/10 | Ростер интересен только при регулярных тратах |
| Конверсия free→paid | 8/10 | Эмоциональный пик = правильный момент для paywall |
| Соответствие RU рынку | 9/10 | Самоирония популярна в рунете; [ВКонтакте viral memes] |

**Risk:** Российская аудитория может воспринять агрессивный ростер как обиду, не как юмор.  
**Mitigation:** Тональность — "дружелюбная честность", а не агрессия. Тест на фокус-группе 20 чел.

### Вариант B: "Финансовый Друг"
**Score: 6.9/10**

| Критерий | Оценка | Обоснование |
|----------|:------:|-------------|
| Вирусный потенциал | 5/10 | Supportive content менее виральный |
| Скорость активации | 5/10 | Долгий онбординг (настройка целей) = риск оттока |
| Долгосрочный retention | 9/10 | Привязка к жизненным целям = высокий LTV |
| Конверсия free→paid | 8/10 | Цели → естественная потребность в автопилоте |
| Соответствие RU рынку | 7/10 | Работает, но не дифференцирует от банковских AI |

**Вывод:** Отличный вариант для upgrade-пути, не для первичного запуска.

### Вариант C: "Паразит-Детектор"
**Score: 6.9/10**

| Критерий | Оценка | Обоснование |
|----------|:------:|-------------|
| Вирусный потенциал | 6/10 | "Нашёл скрытые подписки" — шарится, но одноразово |
| Скорость активации | 9/10 | Немедленный результат (найти подписки) = max активация |
| Долгосрочный retention | 5/10 | После первого скана — что дальше? |
| Конверсия free→paid | 6/10 | После нахождения паразитов → мотивация к оплате слабеет |
| Соответствие RU рынку | 8/10 | Актуально: забытые подписки — боль [vc.ru финансы] |

**Вывод:** Лучший onboarding hook, плохой standalone продукт.

## Решение: Гибрид C→A (с элементами B в Plus)

**Принято:**

```
Шаг 1: Паразит-Детектор (Acquisition screen)
  → "За 2 минуты найдём, куда уходят деньги"
  → Высокая конверсия в регистрацию (60-70%)
  
Шаг 2: Честный Ростер (Core AHA Moment)
  → После сканирования → первый ростер
  → Создаёт вирусный контент и вовлечение
  
Шаг 3: Цели (B-элементы в Клёво Plus)
  → Автосбережения к конкретной цели
  → Удержание платящих пользователей
```

**Обоснование гибрида:**
1. **C снижает барьер входа** — рациональный крюк ("найдём деньги") проще продать, чем юмор
2. **A создаёт вирусность** — без вирусного механизма рост дорогой  
3. **B обеспечивает LTV** — без долгосрочной ценности churn высокий через 3 мес.

**Источники для решения:**
- [FHaaS trend 2026](https://innowise.com/blog/fintech-trends/) — bundled financial wellness
- [Cleo roast mode success](https://overdraftapps.com/cleo-roast-mode/) — 300K roast sessions/year
- [Gen Z self-awareness gap](https://www.researchgate.net/publication/391214267_The_Role_of_Fintech_in_Shaping_the_Spending_and_Saving_Habits_of_Millennials_and_Gen_Z) — знают, но не делают

## Последствия

- **MVP Scope:** Паразит-сканер + первый ростер + топ-5 категорий  
- **v1:** Полный ростер-цикл + шеринг + базовые советы  
- **v2:** Цели + автосбережения (Клёво Plus)  
- **Продуктовые метрики:** Activation rate 55%+, D7 retention 35%+, K > 1.2
